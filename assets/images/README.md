# Assets - Images

Coloque sua imagem de fundo aqui com o nome `background.jpg`.

A imagem será usada como fundo da documentação.

**Recomendações:**
- Formato: JPG, PNG ou WebP
- Tamanho: 1920x1080 ou maior (para melhor qualidade)
- Peso: Otimize a imagem para web (máximo 500KB recomendado)
- Nome do arquivo: `background.jpg`
