# Assets - Logos e Imagens

Esta pasta deve conter os seguintes arquivos de logo/imagem:

## Arquivos necessários:

1. **logo.png** ou **logo.svg**
   - Logo principal do Cellframe
   - Usado no título do sidebar
   - Tamanho recomendado: 24x24px a 48x48px

2. **logo-header.png** ou **logo-header.svg**
   - Logo para a página inicial (README)
   - Tamanho recomendado: 200-300px de largura

3. **favicon.png**
   - Favicon do site (16x16px ou 32x32px)
   - Usado na aba do navegador

4. **favicon.svg**
   - Favicon em formato SVG (opcional, melhor qualidade)
   - Usado na aba do navegador

## Pontos estratégicos onde os logos são usados:

- **Sidebar**: Logo ao lado do título "Cellframe Ambassador Program"
- **Favicon**: Ícone na aba do navegador
- **Página inicial (README)**: Logo no topo da página
- **Apple Touch Icon**: Logo para dispositivos iOS (opcional)

## Formato recomendado:

- PNG: Para logos com transparência
- SVG: Para melhor qualidade e escalabilidade
- Tamanhos: Múltiplos tamanhos para diferentes usos (favicon pequeno, logo maior)
